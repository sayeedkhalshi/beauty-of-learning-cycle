# intentionalism
Every cycle starts with a intention and ends in a non intention to actualize it. Every cycle has a lot of phases and every phase has many micro cycles. Same intentions are making the same thing.

An intentional structure would be controlling the outcome of each cycle as needed. To do that, they would have to learn to figure out the micro cycles inside it. As much as they can find, the better the learner. Then they would have to engineer the changes in a way the outcome is positive. What they build is a control structure.
